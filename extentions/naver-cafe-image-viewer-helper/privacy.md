# 개인정보처리방침 - Naver Cafe Image Viewer Helper

최종 업데이트: 2026년 2월 19일

## 1. 개요
Naver Cafe Image Viewer Helper 확장 프로그램(이하 “본 확장 프로그램”)은  
네이버 카페의 이미지 뷰어를 강화하며,  
사용자의 개인정보를 수집하거나 저장하지 않습니다.

본 개인정보처리방침은 본 확장 프로그램이 정보를 어떻게 처리하는지 설명합니다.

---

## 2. 수집하는 정보
본 확장 프로그램은 다음을 포함한 **어떠한 개인정보도 수집하지 않습니다**.

- 개인 식별 정보
- 계정 정보
- 브라우징 기록
- IP 주소
- 미디어 데이터(오디오/비디오)
- 사용자 생성 콘텐츠

---

## 3. 데이터 처리 방식
본 확장 프로그램의 모든 기능은 **사용자의 브라우저 내에서만 로컬로 수행**됩니다.

- 외부 서버와 통신하지 않습니다.
- 어떠한 데이터도 저장하거나 전송하지 않습니다.

---

## 4. 데이터 공유
본 확장 프로그램은 제3자와 **어떠한 데이터도 공유하지 않습니다**.

수집되는 데이터가 없으므로, 공유되는 데이터 또한 존재하지 않습니다.

---

## 5. 권한 사용
본 확장 프로그램은 기능 제공을 위해 최소한의 권한만을 사용합니다.
- `storage`: 변경된 설정을 저장 하기 위해
- 호스트 권한(`naver.com`): 네이버 카페 이미지 뷰어에서만 기능을 동작시키기 위해 사용됩니다.

해당 권한들은 기능 제공 목적 외의 용도로 사용되지 않습니다.

---

## 6. 쿠키 및 추적
본 확장 프로그램은 쿠키를 사용하지 않으며,  
사용자를 추적하거나 식별하지 않습니다.

---

## 7. 분석 도구
본 확장 프로그램은 **분석 도구(Google Analytics 등)를 사용하지 않습니다**.

---

## 8. 아동의 개인정보
본 확장 프로그램은 개인정보를 수집하지 않으므로,  
모든 연령대의 사용자가 안전하게 사용할 수 있습니다.

---

## 9. 보안
모든 처리는 사용자의 브라우저 내에서 로컬로 수행되며,  
외부 네트워크를 통한 데이터 전송이 존재하지 않습니다.

---

## 10. 개인정보처리방침 변경
본 개인정보처리방침은 기능 변경 또는 법적 요구 사항에 따라 수정될 수 있습니다.  
변경 사항은 확장 프로그램의 배포 페이지 또는 본 문서를 통해 공지됩니다.
